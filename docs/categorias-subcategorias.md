## 🔄 Cómo Usar Este Documento

1. **Para añadir nuevos términos a un sector existente**: Simplemente agrega la línea bajo el sector correspondiente
2. **Para crear un nuevo sector**: Añade un nuevo `### Nombre Sector` con sus términos
3. **Para crear una nueva categoría**: Sigue el formato `## 🔲 N. Nombre (X Sectores)`
4. **Después de modificar**: Actualiza la base de datos en Supabase con el SQL correspondiente

---

**Nota:** Esta estructura es dinámica y se actualiza cada vez que se modifican las categorías en la admin panel de IO Prospector.
